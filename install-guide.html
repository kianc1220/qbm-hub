Install on your phone — quick setup guide
🌐 qbm-operator-hub.netlify.app
Scan to open
🍎
iPhone / iPad
Safari browser required
1
Open Safari
Must use Safari — Chrome on iPhone does not support Add to Home Screen.
2
Go to the website
Type qbm-operator-hub.netlify.app in the address bar and press Go.
3
Tap the Share button
Tap the □↑ Share icon at the bottom of the screen (middle of the toolbar).
4
Add to Home Screen
Scroll down in the share sheet and tap Add to Home Screen.
5
Tap Add
Leave the name as QBM Hub and tap Add in the top right. Done! ✓
🤖
Android
Google Chrome browser required
1
Open Chrome
Use Google Chrome for the best experience and install support.
2
Go to the website
Type qbm-operator-hub.netlify.app in the address bar and press Enter.
3
Tap the install banner
Chrome will show a pop-up at the bottom: Add QBM Hub to Home screen — tap it.
4
Or use the menu
If no banner appears, tap ⋮ Menu (top right) → Add to Home screen.
5
Tap Install
Tap Install or Add in the confirmation dialog. Done! ✓
