QBM Operator Hub
Install on your phone — quick setup guide
🌐 qbm-operator-hub.netlify.app
Scan to open
🍎
iPhone / iPad
Safari browser required
1
Open Safari
Must use Safari — Chrome on iPhone does not support Add to Home Screen.
2
Go to the website
Type qbm-operator-hub.netlify.app and press Go.
3
Tap the Share button
Tap the □↑ Share icon at the bottom centre of Safari.
4
Add to Home Screen
Scroll down the share sheet and tap Add to Home Screen.
5
Tap Add — done! ✓
Leave the name as QBM Hub and tap Add in the top right.
🤖
Android
Google Chrome browser required
1
Open Chrome
Use Google Chrome for the best experience and install support.
2
Go to the website
Type qbm-operator-hub.netlify.app and press Enter.
3
Tap the install banner
Chrome shows a banner at the bottom: Add QBM Hub to Home screen — tap it.
4
Or use the menu
If no banner: tap ⋮ Menu (top right) → Add to Home screen.
5
Tap Install — done! ✓
Tap Install or Add in the confirmation popup.
